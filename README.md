# Weather_app
A simple Python CLI weather application that fetches real-time weather data using **WeatherAPI**

## Features:
- Get current temperature
- Weather condition
- Humidity
- Wind Speed

## Technologies:
- Python
- Requests
- WeatherAPI
- python-dotenv

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/Divnoorheer/Weather_app
cd Weather_app
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Create a .env file
For the api key, please login/create an account on "https://www.weatherapi.com/login.aspx".
```env
WEATHER_API_KEY=your_api_key_here
```

4. Run the program
```bash
python weather_app.py
```
